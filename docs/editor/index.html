Editor¶

Let’s change Flask to automatically reload whenever the Python changes using debug=True.

Features Shown¶

  • Opening files, tabs and tab placement
  • Different ways to open files
  • The “mini-UI” in tool windows

Steps¶

  1. Double-click in Project Tool to open src/main.py
  2. Close tab
  3. Open using Cmd-E, close with Cmd-W
  4. Open using Navigation Bar
  5. Open another file, re-arrange tabs
  6. Split the tabs into two windows
  7. Switch to no tabs, switch back
  8. Edit main.py to set debug=True
  9. Restart the run config, edit the hello message, reload browser
  10. Explain the mini-UI inside the run and project windows
  11. Move tool windows around, float
  12. Put in an error, close the tab, reload browser, click on traceback
