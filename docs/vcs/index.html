Version Control¶

PyCharm has a superb visual frontend to version control.

Features Shown¶

  • Git diff, commit, branch, merge, history
  • Local History, and alternative to Git
  • Editor gutters

Steps¶

  1. Open Version Control and see which files have changed
  2. Diff to see the changes
  3. Look at the gutter to see the changes, right-click to show commit
  4. Close Version Control tab
  5. Ctrl-V, Commit
  6. Make a new branch
  7. Change the returned string
  8. Commit, go back to master
  9. Compare the new branch, then merge, delete branch
  10. Version Control -> Log messages
  11. Show Local History
