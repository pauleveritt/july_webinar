Refactoring¶

Professional tools such as PyCharm can do refactoring work for you, fixing code across your project.

Let’s rename our file, template, and a route.

Features Shown¶

  • Extensive refactorings, such as rename
  • Actions findable by menus, key bindings, and search
  • Rich Python console configured for current project

Steps¶

  1. Make hello.py which imports from main.py
  2. Use refactor to change main.py to app.py
  3. Show that it changed in: VCS, Run Configuration, hello.py
  4. Safe Delete hello.py (and show that it is gone from VCS)
  5. Rename template and route to be todo-focused.
  6. “Actions”: Show the different ways that Refactor Rename can be invoked
  7. Open the Python Console and import a function, then close
  8. Show how preferences configure the console
  9. Install IPython, then go to console
