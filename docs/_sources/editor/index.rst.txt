Editor
======

Let's change Flask to automatically reload whenever the Python
changes using ``debug=True``.

Features Shown
--------------

- Opening files, tabs and tab placement

- Different ways to open files

- The "mini-UI" in tool windows

Steps
-----

#. Double-click in Project Tool to open ``main.py``

#. Close tab

#. Open using Cmd-E, close with Cmd-W

#. Open using Navigation Bar

#. Open another file, re-arrange tabs

#. Split the tabs into two windows

#. Switch to no tabs, switch back

#. Edit ``main.py`` to set ``debug=True``

#. Restart the run config, edit the hello message, reload browser

#. Explain the mini-UI inside the run and project windows

#. Move tool windows around, float

#. Put in an error, close the tab, reload browser, click on traceback
